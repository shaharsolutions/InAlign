פתיחת המקור
1

כניסה ואימות

Supabase Auth מנהל משתמשים, סשנים והרשאות כניסה למערכת.

2

SPA בדפדפן

Vite, HTML, CSS ו-Vanilla JS מספקים ממשק RTL רספונסיבי.

3

ניהול ולמידה

פורטל מנהל לניהול תכנים ומשתמשים, ופורטל לומד להפעלת תכני למידה.

4

Content Player

נגן ייעודי מציג לומדות, סרטונים, מצגות ו-PDF ומתעד זמן שימוש והשלמות.

5

נתונים ודוחות

Supabase שומר תכנים, שיוכים, צפייה והתקדמות, ומאפשר ייצוא ודוחות.

צד לומד

  • דשבורד תכני למידה אישיים
  • סטטוס, ציון, שימוש והתקדמות
  • נגן לומדה, וידאו, PDF ומצגת

שכבת Backend ונתונים

  • profiles, courses, content_type
  • learner_progress וזמן שימוש
  • Storage מאובטח לתכני הדרכה

צד מנהל

  • דשבורד ניהולי
  • ניהול תוכן למידה ומשתמשים
  • מעקב צפייה, שימוש וייצוא דוחות

תרשים אבטחה קיימת מתוך security_review.md

פתיחת המקור

Authentication

Supabase Auth, JWT וניהול סיסמאות מוצפן.

RBAC

Super Admin, Org Admin ולומד, כל אחד עם פעולות מתאימות.
Defense in Depth אכיפה משולבת בדפדפן, ב-Router, ב-UI ובמסד הנתונים.

RLS ו-Multi-Tenancy

PostgreSQL מבודד נתוני ארגונים ומונע גישה לנתונים זרים.

Frontend Guards

Route Guards, הסתרת פעולות רגישות ובידוד תכני למידה ב-iFrame.

הערה: חלק מהעמודים דורשים משתמש מחובר והרשאות מתאימות. אם אין הרשאה, ה-Router יעביר למסך הכניסה או יציג חסימה.
